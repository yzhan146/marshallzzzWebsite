GetThatJob!

GetThatJob! 针对一个目标 JD，帮你更接近面试。

这个工具不只改简历，也会解释修改原因、指出 gap，并准备可能的面试问题。当前产品流程是：上传简历，提供目标 JD，查看修改了什么以及为什么，最后下载修改后的 PDF 草稿。

公开站点当前是入口页。完整 AI 生成版需要服务端保存 API key、解析 PDF 并生成新 PDF，正在迁移到独立后端服务。
View GetThatJob! on GitHub Contact me
1. 上传原始简历

支持 PDF 简历。系统会以原始简历事实为边界，只做重排、压缩、澄清和重新叙事。

2. 提供一个目标 JD

可以是 JD 链接，也可以是粘贴的岗位描述。现在不再要求提交多个 JD 做排序。

3. 下载修改后简历

结果会包含成功率、修改记录、修改原因、gap、面试问题，以及可下载的 PDF 草稿。

What changed

新版产品定义

  • 核心任务从“帮用户筛多个岗位”改成“帮用户针对一个岗位改简历”。
  • 输出优先级从“排名卡片”改成“修改了什么、为什么这样改、是否有事实依据”。
  • 修改后的简历必须保持原始简历语言。中文简历输出中文，英文简历输出英文，其他语言同理。
  • 不允许为了迎合 JD 编造经历、公司、日期、学历、工具、指标或项目。
  • RAG 从旧的地点/薪资偏好库改成了简历证据库和改写约束库。

Output preview

1 面试成功率和岗位匹配等级
2 What changed and why：逐条解释简历哪里改了、为什么改、证据来自哪里

4 当前 gap 和高频面试问题
