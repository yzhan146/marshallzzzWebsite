Project 1

AI Workflows for Research and Presentations

A three-session teaching project on turning a strong source page or report into structured notes, analysis, and a clear 3–5 slide deck.

Audience: ages 16–18 Format: 3 × 1 hour Demo direction: McKinsey Insights → gold / economics

Project goal

Teach an end-to-end workflow

Students learn how one credible source becomes structured notes, analysis, and a presentation-ready mini deck.

Current demo status

Workflow fixed, source flexible

The current version uses placeholders so the teaching flow is easy to see first. Real sources and data points can be inserted later without changing the structure.

Project summary

Start with one source. End with a short deck.

The teaching strategy is demo-first. Instead of starting with AI theory, students first see a complete example: one source page, one structured extraction, one analysis pass, and one short deck.

That makes the workflow easier to understand and easier to copy. The goal is not blind automation. The goal is to show how AI can speed up structure and drafting while human judgment stays in the loop.

Workflow

From source to slides

Step 1

Choose a credible source

Begin with a credible source page, such as a McKinsey Insights article or report page. The current placeholder topic is gold, macro conditions, and investment logic.

Step 2

Extract the useful information

Use AI to pull out the title, topic, core message, key points, useful facts, and any limitation or uncertainty worth flagging.

Step 3

Turn notes into analysis

Turn the extracted material into a repeatable process: identify the claim, group the drivers, note evidence, and mark what still needs human verification.

Step 4

Build a 3–5 slide deck

Turn the analysis into a short deck with a clear message, concise bullets, and one visual idea per slide.

Sessions

Three sessions

Session 1

Extract Information from a Website

Structured input

Students begin with a real source page and learn how to turn it into usable notes. The emphasis is on source quality, extraction, and asking AI for structure instead of a vague summary.

  • Judge whether a source is credible enough to use
  • Extract the core message, key points, and usable facts
  • Compare a weak prompt with a stronger structured prompt
  • Produce a clean note set that can feed the next step
Session 2

Use AI Workflows for Analysis

Repeatable process

Students move from one prompt to workflow thinking. The extracted notes become input for a simple analysis process that groups evidence, identifies drivers, and prepares a presentation angle.

  • Understand the difference between a prompt and a workflow
  • Break analysis into repeatable steps
  • Flag claims that still need human verification
  • Generate a presentation-ready analysis summary
Session 3

Turn Analysis into a 3–5 Slide Deck

Presentation output

Students use the analysis from the previous step to build a small deck with clear slide logic. AI can help produce the first version, but students still need to refine wording, structure, and emphasis.

  • Create a 3–5 slide outline with one purpose per slide
  • Improve generic AI output into clearer communication
  • Add simple visual suggestions for each slide
  • Explain where human judgment matters most in the final result

Core message

AI speeds up structure, not judgment

The real skill is learning how to move from source material to usable output. AI can accelerate extraction, organization, and first drafts, but strong final work still depends on careful human review.
