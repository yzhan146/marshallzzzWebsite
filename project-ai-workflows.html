Project 1

AI Workflows for Research and Presentations

A three-session teaching project on turning information from a website or report into AI-assisted analysis and a clear 3–5 slide deck.

Audience: ages 16–18 Format: 3 × 1 hour Demo direction: McKinsey → gold / economics

Project goal

Teach an end-to-end AI workflow

Students see how a credible source can be transformed into structured notes, analysis, and a presentation-ready mini deck.

Current demo status

Process first, data later

The current version is intentionally built with placeholders so the workflow is clear first. Real source details and data points can be filled in after the final report is chosen.

Project summary

Show the result first, then explain how it was made.

The teaching strategy is demo-first. Instead of starting with AI theory, students first see a complete example: one source page, one structured extraction, one AI-assisted analysis, and one short deck.

This makes the workflow easier to trust and easier to copy. The point is not blind automation. The point is to show how AI can help turn complex material into usable output while keeping human review in the loop.

Workflow

End-to-end demo path

Step 1

Choose a credible source

Start with a structured source page, such as a McKinsey report or insight article. The working placeholder topic is gold, macro conditions, and investment logic.

Step 2

Extract the useful information

Use AI to pull out the title, topic, core message, key points, notable facts, and any limitation or uncertainty worth flagging.

Step 3

Turn notes into analysis

Organize the extracted material into a repeatable workflow: identify the claim, group the drivers, note evidence, and mark which parts still require human verification.

Step 4

Build a 3–5 slide deck

Convert the analysis into a short deck with a clear message, concise bullets, and one visual idea per slide.

Sessions

Three sessions

Session 1

Extract Information from a Website

Structured input

Students begin with a real source page and learn how to turn it into usable notes. The emphasis is on source quality, extraction, and asking AI for clear structure instead of vague summary.

  • Judge whether a source is credible enough to use
  • Extract the core message, key points, and useful facts
  • Compare a weak prompt with a stronger structured prompt
  • Produce a clean note set that can feed the next step
Session 2

Use AI Workflows for Analysis

Repeatable process

Students move from one prompt to workflow thinking. The extracted notes become input for a simple AI-assisted analysis process that groups evidence, identifies drivers, and prepares a presentation angle.

  • Understand the difference between a prompt and a workflow
  • Break analysis into repeatable steps
  • Flag claims that still need human verification
  • Generate a presentation-ready analysis summary
Session 3

Turn Analysis into a 3–5 Slide Deck

Presentation output

Students use the analysis from the previous step to build a small deck with clear slide logic. AI helps create the first version, but students still need to refine wording, structure, and emphasis.

  • Create a 3–5 slide outline with one purpose per slide
  • Improve generic AI output into clearer communication
  • Add simple visual suggestions for each slide
  • Explain where human judgment matters most in the final result

Core message

AI speeds up structure, not judgment

The real skill is learning how to move from source material to usable output. AI can accelerate extraction, organization, and first drafts, but the strongest final work still depends on careful human review.
